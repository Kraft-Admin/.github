# Kraft

**Open-source infrastructure for building enterprise applications.**

Kraft is an open-source initiative focused on helping developers build modern enterprise applications faster. We believe developers should spend less time building repetitive administration features and more time solving business problems.

## Current Project

### KraftAdmin

KraftAdmin is an open-source admin framework for Spring Boot that automatically transforms your entities into a modern administration interface.

Features include:

* Automatic CRUD generation
* Modern responsive admin interface
* Pagination, filtering and search
* Entity relationships
* Authentication & authorization support
* Extensible architecture
* Spring Boot integration

> ⚠️ **Early Development**
>
> KraftAdmin is under active development and is not yet production-ready. Features and APIs may change before the first stable release.
